# ZipUp

## 프로젝트 개요

프론트엔드와 백엔드을 공부하기 위한 간단한 온라인 쇼핑몰 팀 프로젝트입니다.  
Spring Boot를 기반으로 백엔드를 구축하며 HTML, CSS, JavaScript로 프론트엔드를 구성합니다.

## 주요 기능

1. **회원가입 및 로그인**

   - 사용자와 판매자로 역할(role) 구분

2. **상품 관리**
   - **상품 리스트** : 메인 페이지에서 카테고리별 상품 보기
   - **상품 상세** : 이미지, 상품명, 가격, 옵션(사이즈, 색상 등), 상품 설명 표시
   - **상품 등록** : 판매자 전용 기능
